# Night OWL Audio Tracks

Place licensed ambience files in this folder.

## Calm Track

Requested source:

https://freesound.org/people/kyles/sounds/452094/

License: Creative Commons 0

Because Freesound requires login for the original FLAC download, download it manually from Freesound, then save it here as one of these names:

```text
public/audio/calm.flac
public/audio/calm.mp3
public/audio/calm.ogg
```

Night OWL will automatically use it for the Calm forest mode. If the file is not present, the app falls back to the built-in procedural calm forest ambience.
